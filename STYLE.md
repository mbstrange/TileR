# Style Guide Rules

### Classes
- **UpperCapitalCamel**
- `class NewClass { }`

### Functions
- Public/Protected
	- **UpperCapitalCamel**
	- `void GetInfo();`
- Private
	- **lowerCamelCase**
	- `void internalHelper();`

### Variables

All **lowerCamelCase**

- Constants
	- **ALL_CAPS**
- Pointers
	- start with **p**
	- `pValue`
- Member of classes
	- start with **m**
	- `mValue`
	- if pointer then `mpValue`
- Globals
	- start with **g**
	- `gGlobalValue`
- Static
	- start with **s**
	- `sValue`
